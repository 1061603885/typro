#### 为什么使用Redis

```
硬盘读写速度太慢，会导致数据库性能有瓶颈
因此需要使用缓存   Redis
作用:
	1.帮助数据库拦截请求
	2.防止缓存穿透(布隆算法)

缓存使用的是内存，Redis既使用内存，也使用硬盘(硬盘做数据备份)
```



#### 安装后修改配置文件

```
打开Redis服务配置文件
修改Redis.windows-service.conf文件
	找到含有# requirepass foobared字样的地方，追加输入
	requirepass 密码
取消限制本地访问
	找到 #bin 127.0.0.1 ::1 没有#则加上
	
```
