#### this指向

```js
1.通过对象调用函数，this指向对象
2.直接调用函数，this指向window
3.如果通过new调用函数，this指向新创建的对象
4.如果通过apply,call,bind调用函数，this指向指定的数据
5.如果是DOM事件函数，this指向事件源
```



#### 箭头函数

```js
箭头函数是一个函数表达式，理论上，任何使用函数表达式的场景都可以使用箭头函数

箭头函数中的this指向，取决于函数定义位置的this指向，而与在哪里调用无关 


如果想直接返回一个对象
const sum = (a,b) => ({
    a:a,
    b:b,
    sum:a+b
})


*箭头函数中，没有自己的this、arguments、new.target,如果使用了，拿到的是函数外层的对应数据
```
