#### Taro下载安装

```react
yarn global add @tarojs/cli

//如果出现sass相关错误,可以使用
yarn global add mirror config-china


//初始化项目
taro init 项目名

//不在全局安装的情况下创建模板项目
npx @tarojs/cli init myApp

//使用 yarn 安装依赖
```





#### 微信小程序

```js
 yarn dev:weapp		//开发环境
 yarn build:weapp	//生产环境(打包)
```



#### 百度小程序

```js
yarn dev:swan
yarn build:swan
```



#### 支付宝小程序
