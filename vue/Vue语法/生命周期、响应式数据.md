#### 生命周期

```js
创建
beforeCrete     //实例初始化守，数据关侧和事件配置之前被调用
created         //实例对象创建之后被调用
挂载
beforeMount     //在挂载开始之前被调用
mouted			//el被新创建的vm.$el替换，并且挂载到实例上去
更新
beforeUpdate 	//数据更新时调用，发生在虚拟DOM打补丁之前
updated			//数据更改会导致虚拟DOM重新渲染和打补丁,在这之后调用函数
销毁
beforeDestory	//实例销毁之前调用
destoryed		//实例销毁之后调用
```
